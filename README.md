# 📝 SecFolio - Global Resume Platform

**SecFolio**에 오신 것을 환영합니다!
이름과 직무를 입력하여 나만의 멋진 이력서를 만들고, 글로벌 IT 기업의 HR 담당자에게 직접 입사 지원을 해보세요. 
저희 HR 봇은 여러분이 제출한 이력서를 24시간 언제나 신속하게 검토합니다.

## 🎯 Objective (목표)
**SecFolio**에 숨겨진 **FLAG (WSL{...})** 를 획득하세요!
 
## 🚀 How to Run 
Docker와 Docker Compose가 설치된 환경에서 아래 명령어를 실행하여 문제 환경을 구축합니다.

```Bash
docker-compose up -d
```

- 웹 서비스 접속: http://localhost:3000
